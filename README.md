## Setup level bot
Create a .env file and copy this code with your corresponding values:

```
DISCORD_TOKEN=YOUR_DISCORD_TOKEN
LEVEL_CHANNEL_ID=YOUR_LEVEL_CHANNEL_ID
GUILD_ID=YOUR_SERVER_ID
```
